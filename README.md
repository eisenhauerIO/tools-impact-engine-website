*Project website for the Impact Engine ecosystem — value proposition, methodology, architecture, and memos*

The Impact Engine documentation lives across component-level Sphinx sites. But none of them
answer the first question a visitor has: why does this exist, and why should I care?

This static site fills that gap. It covers the problem (index), the science (causal inference,
evidence scoring, decision theory), and the system architecture — plus a growing collection of
memos on topics like validation strategy, using Impact Engine as the running example.

Visit [eisenhauerio.github.io/tools-impact-engine-website](https://eisenhauerio.github.io/tools-impact-engine-website).
